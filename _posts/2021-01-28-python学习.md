## 1. 变量的格式化输出

| 格式化字符 |                             含义                             |
| :--------: | :----------------------------------------------------------: |
|     %s     |                            字符串                            |
|     %d     | 有符号十进制整数，如`%06d` 表示输出的整数显示位数为`6` ，不足的地方使用 `0` 补全，如`000001`,整数大于6位时全部显示 |
|     %f     |            浮点数，`%.2f` 表示小数点后只保留两位             |
|     %%     |                           输出`%`                            |

- 语法格式

  ```python
  print("格式化字符串" % 变量1)
  
  print("格式化字符串" % (变量1, 变量2...))
  ```
